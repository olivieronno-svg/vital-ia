# VITAL I.A — Site légal & Politique de confidentialité

Site web officiel hébergeant les **Conditions Générales d'Utilisation** et la **Politique de confidentialité** de l'application **VITAL I.A** (anciennement BilanAmbu).

## 🩺 À propos de VITAL I.A

VITAL I.A est l'assistant médical intelligent pour les gestes qui sauvent.

**Fonctionnalités principales** :
- 🚑 **Tutos gestes d'urgence** guidés par I.A vocale (RCP/DAE + 16 gestes : PLS, étouffement, hémorragie, AVC, anaphylaxie, etc.)
- 💓 **Tuto ECG 12 dérivations** selon protocoles CESU / AFRTAL
- 📋 **Bilan ambulancier** structuré XABCDE avec mannequins anatomiques zoomables
- 💨 **Calculateur d'autonomie O₂** avec manomètre tactile
- ⏱ **Chronomètre persistant**, FC/FR tactiles, 7 scores cliniques
- 🗺 **Géolocalisation** des DAE et hôpitaux d'urgence
- 📅 **Calendrier de recyclages** (AFGSU, PSE, etc.)
- 🌍 Disponible dans **14 langues**

**Audience** : ambulanciers, secouristes, infirmiers, professionnels de santé, et grand public formé aux premiers secours.

VITAL I.A est un outil pédagogique et d'aide à la saisie. Il **ne remplace pas** une formation officielle aux premiers secours (PSC1, AFGSU, PSE), ni le jugement clinique d'un professionnel de santé, ni les protocoles de votre service.

En cas d'urgence vitale, appelez **15 / 18 / 112** (France) ou le numéro d'urgence local de votre pays.
